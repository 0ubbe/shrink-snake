## The Story

This game was built from a single sentence: *"Snake but the arena shrinks."*

An AI read those six words, wrote every line of code, designed the visuals, deployed it to the internet, and wrote the page you're reading now — all in one conversation, in under two minutes. No frameworks. No build tools. No human wrote a single character of code.

The result is a complete, polished browser game that fits inside a single HTML file.

## The Game

You're a snake. You eat. You grow. You know the drill.

But every 4 seconds, the walls close in. The arena shrinks. The space you have to survive gets smaller, tighter, more claustrophobic. The longer you last, the faster you move and the less room you have. Eventually, the walls come for you.

**How long can you survive the shrink?**

## What's Inside

- Canvas 2D rendering with buttery 60fps animation
- Dynamic particle systems for food collection, wall compression, and death
- Screen shake with intensity decay
- Glowing neon aesthetic with starfield background
- Danger-zone indicators when the walls get tight
- Progressive difficulty — speed increases with each bite
- Local high score persistence via `localStorage`
- Fully responsive — adapts to any screen size
- Touch + keyboard input with 180° turn prevention
- Zero dependencies. One file. ~350 lines of vanilla JS.

## Run Locally

```bash
git clone https://github.com/0ubbe/shrink-snake.git
cd shrink-snake
open index.html
```

That's it. No `npm install`. No build step. Just open and play.
